ChocAn System – Group 4

Description:
The ChocAn system is a software application designed to simulate member validation, service recording, provider directory access, and weekly report generation for healthcare providers.

System Requirements:
- Python 3.x installed
- Terminal or command prompt (Mac Terminal, Windows Command Prompt, or VS Code terminal)

How to Run the Program:
1. Navigate to the project folder containing chocan.py
2. Run the following command:

   python3 chocan.py

Program Features:
- Member validation (valid, invalid, suspended)
- Service recording with validation
- Provider directory display
- Weekly report generation of recorded services

Notes:
- This is a simplified implementation using in-memory sample data
- Some advanced features such as database storage, operator maintenance functions, and external system integrations are not implemented
- Edge cases such as empty directory and full input validation are partially implemented

File Structure:
- chocan.py → main program file
- README.txt → instructions for running the program
